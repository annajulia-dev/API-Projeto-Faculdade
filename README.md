# API em NodeJS Projeto da faculdade

## Tecnologias
- NodeJS
- Express
- TypeScript
- MongoDB

## Arquitetura
Monólito modular com camadas

## Etapas para a criação da API

- Criar repositório e configurar projeto pc
- Iniciar um projeto Node (comando)
- Libs
    - Express
    - type de Express
    - type do Node
    - typescript
    - tsx
    - dotenv
    - cors
    - type cors
- Criar o arquivo tsconfig
- Vão ajustar o tsconfig de acordo com o projeto
- Montar a arquitetura: monolito modular com camadas.
- Criar o npm run dev
- Criar e configurar arquivo app.ts
- Codar route.ts
- Codar server.ts
- Codar .env
- Criar rota de teste e fazer uma requisição postman
